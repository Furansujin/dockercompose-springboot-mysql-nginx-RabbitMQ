# [Docker Compose with Spring Boot, Inmemory, rabbitmq]
## What you'll build
- A simple Spring Boot application with iMInmemory, rabbitmq running inside Docker containers

## What you'll need
- Docker CE

## Stack
- Docker
- Java
- Spring Boot
- RABBIRMQ
- Maven

## Run
- Run command `docker-compose up`
- Access to http://localhost/
